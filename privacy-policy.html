Privacy Policy – JetWages

Last updated: June 1, 2025

1. Local-Only Data Handling

JetWages is built to work entirely offline. All personal data — including schedules, payslips, pay information, and settings — is stored locally on your device using secure local storage (AsyncStorage).

2. No External Data Transmission

JetWages does not send any data to external servers. No cloud sync, analytics tracking, or third-party APIs are involved in processing or storing your data.

3. File Processing

The app allows importing HTML and PDF files from eCrew and supplementary payslips. These are processed entirely on-device using expo-file-system and memory-only operations. No files or extracted data leave your device.

4. Local Analytics (Optional)

JetWages tracks setup progress, file uploads, and historical pay analysis. These analytics:

  • Are stored only on your device
  • Do not contain personal information
  • Can be exported via the Settings screen

5. Backup & Restore

You may export or import your JetWages data for backup. These operations are handled locally via the device’s file system or sharing tools. No external services are involved.

6. External Libraries

We use a few open-source libraries (e.g., for parsing PDF files) that are bundled inside the app and run locally. These libraries do not collect or transmit any user data.

7. User Control

You have complete control over your data. Uninstalling the app permanently deletes all stored content. JetWages never stores or transmits any of your data to third parties.

8. Contact

If you have questions about this policy, contact us at jetwages.app@gmail.com
